# 多维表格字段说明

## 任务表（默认表）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 任务名称 | 文本 | 任务的简短描述 |
| 负责人 | 人员 | 飞书用户，包含 open_id 和名称 |
| 状态 | 单选 | 待开始 / 进行中 / 已完成 / 已超期 |
| 计划完成日期 | 日期 | 格式 YYYY-MM-DD |
| 实际完成日期 | 日期 | 完成后填写 |
| 优先级 | 单选 | 高 / 中 / 低 |
| 备注 | 文本 | 补充说明 |

## 候选人评估表

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 候选人姓名 | 文本 | |
| 目标岗位 | 文本 | |
| 综合评分 | 数字 | 0-100 |
| 推荐结论 | 单选 | 推荐约面试 / 暂不推荐 / 待补充材料 |
| 评估日期 | 日期 | |
| 审批状态 | 单选 | 待审批 / 审批中 / 已通过 / 已拒绝 |
| 评估报告 | 文本 | 完整评估内容 |
