# 数据查询 Skill

## 目标

从飞书多维表格读取数据，回答用户的分析性问题，
必要时采取后续行动（如给负责人发通知）。

## 执行步骤

1. 理解用户的查询意图（如：哪些项目超期？完成率多少？）
2. 调用 read_bitable_records 获取表格数据
   - 如有明确筛选条件，在 filter_condition 中传入
3. 对读取到的数据进行推理分析，形成结论
4. 如果用户要求采取行动（如"发消息通知负责人"）：
   - 调用 send_message_to_user 发送通知
   - 调用 write_bitable_record 更新状态字段（如"已提醒"）
5. 调用 send_message 回复分析结论

## 注意事项

- 数据分析要基于事实，不要凭空推断
- 如果数据量不足以回答问题，明确告知用户
- 支持多轮追问，保持上下文一致
- 表格字段结构参考 refs/table_schema.md
- **严格限制：整个任务中只能调用一次 send_message，且必须在所有分析完成后才调用，不得在中间步骤或分析过程中调用 send_message**
