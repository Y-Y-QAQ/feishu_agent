# 招聘周报生成 Skill

## 目标

从飞书多维表格读取候选人评估数据，统计本周招聘工作进展，
自动生成结构化招聘工作周报，发送到当前会话。

## 执行步骤

1. 调用 read_bitable_records 读取候选人评估表中的全部记录
2. 从记录中统计以下关键数据：
   - 本周新增评估候选人数量及姓名
   - 各岗位候选人分布（AI工程师 / 飞书开发工程师 / 其他）
   - 推荐约面试人数、暂不推荐人数、待补充材料人数
   - 高分候选人（评分 >= 85）重点介绍
   - 审批流程进展（审批中、已通过、已拒绝各几人）
3. 如果用户在消息中有额外补充（如下周招聘计划、需要协调资源），一并纳入
4. 使用 weekly_report_template 模板生成招聘周报正文
5. 调用 send_message 发送到当前飞书会话

## 注意事项

- 周报语气专业简洁，数据准确，重点突出高潜候选人
- 如果表格数据为空，告知用户"本周暂无候选人评估记录"
- 支持用户追问修改，如"把高分候选人部分写详细一点"，则重新生成并发送
- 表格字段含义参考 refs/table_schema.md
- **严格限制：整个任务中只能调用一次 send_message，且必须在所有步骤完成后才调用，不得在中间过程中调用 send_message**
