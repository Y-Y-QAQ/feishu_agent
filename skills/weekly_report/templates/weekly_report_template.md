# 招聘工作周报模板

```
招聘工作周报
周期：{本周起止日期，如 2026-06-15 至 2026-06-21}
汇报人：招聘助手 Agent

一、本周招聘概况

- 本周累计评估候选人：{N} 人
- 推荐约面试：{N} 人
- 暂不推荐：{N} 人
- 待补充材料：{N} 人

二、重点候选人进展

{列出评分 >= 85 的候选人，每人一条，格式：
姓名 | 目标岗位 | 综合评分 | 推荐结论 | 审批状态}

三、各岗位候选人分布

{按岗位分组，列出候选人数量及平均分}

四、审批流程跟进

{审批中 N 人，已通过 N 人，已拒绝 N 人，具体说明待跟进事项}

五、下周招聘计划

{如用户有补充则列出，否则填写默认计划：继续推进高分候选人面试，补全待补充材料候选人信息}

六、需要协调资源或支持

{如有，说明；如无，填写"暂无"}
```
