# 候选人评估表字段说明

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 候选人姓名 | 文本 | 候选人的真实姓名 |
| 目标岗位 | 文本 | 应聘的岗位名称 |
| 综合评分 | 数字 | AI 评估综合得分，0-100 分 |
| 推荐结论 | 单选 | 推荐约面试 / 暂不推荐 / 待补充材料 |
| 评估日期 | 日期 | 完成评估的日期，毫秒时间戳 |
| 评估报告 | 文本 | AI 生成的详细评估内容 |

## 推荐结论说明

- **推荐约面试**：综合评分 >= 80，各维度表现良好
- **暂不推荐**：与岗位需求差距较大，不进入面试流程
- **待补充材料**：有潜力但信息不足，需候选人补充说明

## 评分标准

- 85 分以上：优质候选人，重点推进
- 75-84 分：良好，可安排面试
- 60-74 分：一般，需补充材料再评估
- 60 分以下：不推荐
